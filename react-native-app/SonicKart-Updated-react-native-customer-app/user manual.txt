1) PROJECT SUMMARY
====================================================================
This is a React Native mobile app for SonicKart with customer-side grocery ordering and package delivery flows.

Main active user journey:
- Customer login with phone number and agreement checkbox.
- Browse categories and products.
- Add to cart and place order (COD or Online via Razorpay).
- Track active product order live on map.
- Manage profile and address book.
- Create package orders (send/receive flow) and track package details.

Delivery-partner code exists, but delivery routes are currently not mounted in main navigation.

11) BUILD / RUN COMMANDS
====================================================================
From project root:
- Install deps: npm install (or yarn install)
- Start metro: npm run start
- Android debug: npm run android
- iOS debug: npm run ios
- Lint: npm run lint
- Tests: npm run test
- Release APK + AAB (Windows):
  cd android
  .\gradlew.bat clean assembleRelease bundleRelease
  Outputs:
  - android/app/build/outputs/apk/release/app-release.apk
  - android/app/build/outputs/bundle/release/app-release.aab

- Optional Windows workaround for long-path CMake issues:
  - Build from short path clone/copy (example: C:\scbuild) if React Native reanimated CMake path errors occur.

- 16KB verification (APK):
  - zipalign -c -P 16 -v 4 android/app/build/outputs/apk/release/app-release.apk
  - Expect: Verification successful
